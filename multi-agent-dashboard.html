核心层 / Core Layer

协调、调度、审查 — 系统的中枢神经系统

🎯

Coordinator Agent

协调Agent — 任务接收与用户交互
profile: coordinator 不执行具体工作
  • 与用户对话确认任务细节
  • 生成结构化任务描述
  • 发布到 Kanban 系统（带类型标签）
  • 跟踪进度并汇总最终报告
  • 铁律：❌ 不执行任何具体任务
SOUL.md: 3.3KB 状态: ✅ 已配置
📡

Dispatcher Agent

调度Agent — 类型匹配与任务分配
profile: dispatcher type:code → dev type:research → researcher
  • 轮询 Kanban ready 状态任务
  • 解析类型标签（title 前缀 + body 标签）
  • 从专家库匹配最适合的 Agent
  • 处理专家拒绝的任务
  • 铁律：❌ 不执行具体任务
SOUL.md: 4.1KB 状态: ✅ 已配置
🔍

Reviewer Agent

审查Agent — 质量验收与反馈循环
profile: reviewer 审查清单 最多 3 次修改循环
  • 验收完成的任务（对照验收标准）
  • 拒绝时重新打类型标签
  • 不通过时提供具体反馈
  • 创建修改任务（最多 3 次循环）
  • 铁律：❌ 不修改代码，只评论建议
SOUL.md: 5.2KB 状态: ✅ 已配置
